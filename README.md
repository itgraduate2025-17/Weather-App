

# Weather App — Real-Time Weather Dashboard

A modern, responsive weather web app that fetches real-time weather data using the OpenWeatherMap API.  
Built with HTML, CSS, and JavaScript, this project shows strong frontend design, API integration, and clean code practices

---

## Features

- Search by city name or country to get current weather data.
- Use My Location button to fetch weather using your device’s location.
- Switch between Celsius (°C) and Fahrenheit (°F).
- Shows local time, sunrise, and sunset based on location.
- Displays detailed stats like humidity, pressure, and wind speed.
- Responsive design that works on both desktop and mobile.
- Simple, modern glassmorphism-style user interface.
- Includes error handling and loading states for better user experience.

---

## Technologies Used

| Layer | Tools / Frameworks |
|-------|--------------------|
| Frontend | HTML5, CSS3, JavaScript (ES6) |
| API | [OpenWeatherMap API](https://openweathermap.org/api) |
| Design | Minimal, clean glassmorphism UI |
| Layout | CSS Grid and Flexbox |
